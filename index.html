SIGNALLAB v2.0 — HOW TO USE
▸ CTS / DTS MODE
Toggle between Continuous-Time x(t) and Discrete-Time x[n] using the tabs at the top center.
▸ DRAW TAB
Draw signals directly on the canvas. Line Seg (default) draws straight segments by click-drag. Freehand follows your mouse. Ramp / Step / Impulse click to place at that x position. Undo removes the last stroke. Analyze derives the x(t) = u(t)/r(t) equation from your drawing. Overlapping x-ranges between strokes are rejected automatically.
▸ EQN→GRAPH TAB
Type any math expression to plot it. Click the signal chips to append them with + (they accumulate). Supports: u(t), r(t), δ(t), sin, cos, exp, rect, tri, sgn, sinc and arbitrary JS math. Set t start/end to control the plot range.
▸ TRANSFORM TAB
Applies A · x(at + b) to the current signal. Set a = time scale, b = time shift, A = amplitude scale. Use Quick Transform buttons for one-click common operations. Original signal is shown dimmed; transformed in yellow.
▸ DUAL OPS TAB
Operate on two signals: Add, Subtract, Multiply, or Convolve. Choose Eqn Form to type expressions, or Draw Graph to draw x₁ and x₂ directly on the canvases. In Draw mode, select which signal to edit using the Edit x₁ / x₂ buttons.
▸ VIEW CONTROLS
Zoom In/Out, Reset view, toggle Grid. Set custom x-range and y-range and click Apply.
▸ COPY ⎘ & EXPORT
The ⎘ button copies only the RHS of the equation to clipboard. ↓ Export downloads the main canvas as a PNG image. ⎘ Copy Eq in the graph header copies the full equation.
